# Как обновлять сайт

Главный редактируемый файл:

- `content/site-content.js`

Что в нём менять:

- `site`: заголовок страницы, описание, название, ссылка на резюме
- `navigation`: пункты меню на главной
- `hero`: первый экран
- `about`: тексты блока "Обо мне"
- `experience`: опыт работы
- `works`: карточки кейсов на главной
- `contacts`: email, Behance, Telegram и другие контакты

Как вносить изменения:

1. Откройте `content/site-content.js`
2. Найдите нужный блок
3. Измените текст, ссылку или добавьте новый объект в массив
4. Сохраните изменения
5. Закоммитьте и отправьте их в GitHub

Что пока редактируется отдельно:

- тексты самих кейсов:
  - `work/documents-in-deal/index.html`
  - `work/paid-booking/index.html`
- стили:
  - `styles.css`

Если хотите, дальше можно вынести в отдельный контент-файл и сами кейсы, чтобы весь текст сайта жил в одном месте.
